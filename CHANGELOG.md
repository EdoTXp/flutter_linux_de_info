# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-11-03

- Fix Format for C++ files.

## [0.1.0] - 2025-11-03

- Initial implementation of the `flutter_linux_de_info` plugin.
- Created the `getDesktopEnvironment()` method to retrieve the current Linux Desktop Environment from the `XDG_CURRENT_DESKTOP` environment variable.
- Implemented the native C++ code for the Linux platform.
- Included a basic example application to demonstrate the plugin's functionality.
- Added comprehensive `README.md` files for both the main plugin and the example application.
- Removed unused `flutter_linux_de_info_plugin_private.h` header file.
